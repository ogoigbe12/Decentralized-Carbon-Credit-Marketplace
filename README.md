# 🌱 Decentralized Carbon Credit Marketplace

A transparent blockchain-based platform for issuing, trading, and verifying carbon credits using Clarity smart contracts with IoT integration for real-world carbon offset verification.

## 🚀 Features

- 🏭 **Project Registration**: Register carbon offset projects with IoT device integration
- 🔍 **IoT Verification**: Submit real-world verification data from IoT sensors
- 💳 **Credit Issuance**: Issue verified carbon credits for approved projects  
- 🛒 **Marketplace Trading**: List and purchase carbon credits in a decentralized marketplace
- ♻️ **Credit Retirement**: Permanently retire credits to claim carbon offsets
- 📊 **Balance Tracking**: Track user carbon credit balances
- 🔐 **Authorized Verifiers**: Role-based verification system

## 📋 Contract Functions

### Project Management
- `register-project` - Register a new carbon offset project
- `verify-project` - Verify a project (authorized verifiers only)
- `get-carbon-project` - Get project details

### IoT Integration  
- `submit-iot-verification` - Submit IoT sensor data for verification
- `get-iot-verification` - Retrieve IoT verification data
- `authorize-verifier` - Authorize IoT data verifiers (owner only)
